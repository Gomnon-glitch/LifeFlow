💡 Aide — Proxy CORS (si nécessaire)

Si la connexion échoue avec une erreur CORS, crée un proxy Cloudflare Worker gratuit :

  1. Va sur dash.cloudflare.com → Workers & Pages → Create Worker
  2. Remplace le code par :
export default {
  async fetch(request) {
    const url = new URL(request.url);
    const target = request.url.replace(url.origin + '/', '');
    const res = await fetch(target, {
      method: request.method,
      headers: request.headers,
      body: request.method !== 'GET' ? request.body : undefined,
    });
    const r = new Response(res.body, res);
    r.headers.set('Access-Control-Allow-Origin', '*');
    r.headers.set('Access-Control-Allow-Headers',
      'Authorization, Content-Type, Accept');
    return r;
  }
};

3. Déploie et copie l'URL du worker dans le champ "Proxy CORS" ci-dessus.
