📲 Installer LifeFlow sur ton téléphone

⚠️ Pré-requis important

Pour installer LifeFlow comme une application sur ton Pixel, les fichiers doivent être servis via un serveur web (pas juste ouverts en tant que fichier local). Deux options :

Option A — GitHub Pages (gratuit)
Pousse les fichiers sur un repo GitHub → active GitHub Pages → accède à l'URL depuis ton Pixel.

Option B — Serveur local (réseau WiFi)
Sur ton PC, lance python -m http.server 3000 dans le dossier Lifestyle, puis accède à http://[IP-PC]:3000 depuis ton Pixel (même WiFi).

📱 Étapes — Android (Pixel)

1

Ouvre Chrome sur ton Pixel et va à l'adresse de LifeFlow (GitHub Pages ou IP locale)

2

Attends le bandeau "Ajouter LifeFlow à l'écran d'accueil" qui apparaît en bas. Si il n'apparaît pas : tape les 3 points ⋮ en haut à droite

3

Sélectionne "Installer l'application" ou "Ajouter à l'écran d'accueil"

4

Confirme → L'icône LifeFlow ⚡ apparaît sur ton écran d'accueil !

5

Active les notifications → Ouvre l'app → onglet Config → section 🔔 Notifications → "Activer les notifications"

💻 Étapes — PC (Chrome / Edge)

1

Ouvre la page LifeFlow dans Chrome ou Edge

2

Clique sur l'icône 📲 dans la barre d'adresse (à droite) ou va dans les 3 points ⋮ → Installer LifeFlow

3

Confirme l'installation → LifeFlow s'ouvre comme une fenêtre dédiée sans barre d'URL !
