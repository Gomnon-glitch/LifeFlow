# Amélioration du Planning : Dynamisme et Synergie LifeFlow

## 1. Planification "Vivante"
- **Synchronisation en Temps Réel** : Le bandeau "Maintenant" du dashboard affiche l'activité exacte du planning selon l'heure actuelle, au lieu d'une plage horaire fixe.
- **Validation Active** : Possibilité de "Taper" (valider) un créneau une fois terminé pour marquer son exécution.

## 2. Gamification & Streaks
- **Planning Streak** : Nouveau compteur de jours consécutifs où le planning a été respecté (ex: au moins 3 slots validés par jour).
- **Buffs RPG** : 
    - Valider un créneau "Course" → +20% Vitesse d'attaque au héros pendant 4h.
    - Valider un créneau "Japonais" → +15% Puissance des sorts (Mana) pendant 4h.
- **Réduction de Dégâts** : Un score de "Fidélité au Planning" élevé sur la semaine protège le héros RPG contre les boss.

## 3. Connexion au Quotidien
- **Ancrage d'Habitudes** : Lier une habitude (ex: "Hydratation") à un créneau. Valider le créneau dans le planning valide l'habitude.
- **Auto-Log** : Valider une séance de sport dans le planning pré-remplit automatiquement le journal de bord.

## 4. Design & UX
- **Effet Cyber-Focus** : Le créneau actuel dans la grille hebdomadaire est mis en avant avec une animation (glow néon).
- **Conseils du Héros** : Le héros RPG intervient dans l'onglet planning pour encourager la validation des créneaux stratégiques.
